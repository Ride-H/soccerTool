# RPD-X 運用ガイドライン

高校サッカーでの実運用に向けた操作ガイド（PDF）です。自チームのシナリオ作成 → 試合前の手修正 → 保存/共有 → 試合中のリアルタイム戦術変更 → 試合後の振り返り、までを実画面つきで解説します。

- **[RPD-X_運用ガイドライン_v1.3.0.pdf](RPD-X_運用ガイドライン_v1.3.0.pdf)** — ツール本体 v1.3.0 対応

## 版について
ガイドの版はツール本体（`RPDX.VERSION` / Git タグ）と同じ版に紐付きます。全手順は各版と同一コードの実ブラウザ自動操作で動作検証済み（巻末に検証記録・掲載画像は検証時の実画面）。

ツール本体は [ライブデモ](https://ride-h.github.io/soccerTool/) からインストール不要で試せます。
